Карта Meshtastic Россия

Автор: Liam Cottle, перевод и доработка: Владимир Гришин

👋 Добро пожаловать на open source карту узлов (нод) сети Meshtastic (через MQTT).
Возможности
  • На карте отображаются ноды, которые отправили корректную позицию через MQTT.
  • Пакеты с координатами должны быть незашифрованы, либо зашифрованы ключом по-умолчанию.
  • Используйте поиск, чтобы найти ноды по их ID или по имени.
  • Наведите мышкой на ноду (на компьютере), чтобы увидеть основные данные.
  • Кликните мышкой на ноду, чтобы увидеть дополнительную информацию (телеметрию, графы, трассировки).
  • Используйте панель слоев справа вверху, чтобы увидеть соседей и точки маршрута (waypoints).
  • Используйте кнопку настроек и подстройте карту под себя.
  • Нашли ошибку или есть идеия? Добавьте issue на GitHub.
FAQ
  • Как мне добавить моё устройство на карту?
  • Ваше устройство, либо устройство, которое вас слышит, должно выгружать данные на наш сервер MQTT.
  • Пакеты с координатами должны быть незашифрованы, либо зашифрованы ключом по-умолчанию.
  • Если ваше устройство имеет прошивку v2.5 или новее, вам нужно включить "OK to MQTT" в настройках LoRa.
  • Для отправки ваших данных на карту, вам нужно подключиться к MQTT серверу с данными ниже.
Какой MQTT сервер указать в настройках?
  • Адрес: mqtt.meshekb.ru
  • Имя пользователя: meshekb
  • Пароль: large2cats
  • Шифрование включено: Да
  • Вывод JSON: Нет
  • TLS включен: Нет
Советвуем настраивать только специальные выделенные ноды для загрузки данных на сервер (карту).
  • Как мне убрать моё устройство с карты?
  • Устройства, которые не были в сети 7 дней удаляются автоматически.Отключите отправку координат на устройстве, чтобы больше не оказаться на карте.
  • Используйте индивидуальные ключи шифрования, чтобы никто не мог увидеть ваши данные.
  • Если ваше устройство имеет прошивку v2.5 или новее, сервер будет игнорировать ваши данные, если у вас выключена опция "OK to MQTT" в настройках LoRa.
  • Для немедленного удаления вашего устройства с карты напишите мне.
Как мне увидеть соседей, которые видны моему устройству?
Откройте панель слоев справа вверху и включите показ соседей.
Некоторые соседи получены через MQTT, это исправлено в последней прошивке.
  • Почему моё устройство отображается не на том месте?
  • Эта карта "округляет"" ваши координаты (уменьшает точность) для прошивок v2.4 и старее.
  • Для устройств с прошивкой v2.4 координаты округляются до двух знаков после запятой.
  • Для устройств с прошивкой v2.5 и новее, с включенной опцией "OK to MQTT", будет показана позиция с точностью, выбранной вами в настройках (precision).
  • Для устройств с прошивкой v2.5 и новее, с выключенной опцией "OK to MQTT", позиция показана не будет.

Настройки

Настройки сохраняются только в этом браузере.
Ноды, которые не обновлялись указанное время, будут скрыты. Обновите для применения параметра.
Ноды, которые не отправляли данные на MQTT-сервер больше указанного времени, будут отображаться синей точкой.
Ноды, которые не обновлялись указанное время, будут отображаться красной точкой. Обновите для применения параметра.
Точки маршрута старее будут скрыты. Обновите для применения параметра.
Ноды дальше указанного растояния будут скрыты. Обновите для применения параметра.
